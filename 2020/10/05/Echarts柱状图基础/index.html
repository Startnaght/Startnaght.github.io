个名词解释

xAxis 相当于一个直角坐标系的x轴

   -type 声明类型
   -category 也就是要声明的类型
   -data 就是x轴的数值

yAxis 相反也是就是相当于直角坐标系的y轴

   -value 声明这个值
  -series
   —data 就是y轴的数据 当然数据后都会以英文逗号分隔
   —type:”bar” 声明这个是柱状图
   — showBackground: true 补全数据不全的地方 可以理解为没有数据为了好观看而白色标识 只有一个参数
-backgroundStyle: {
color: ‘rgba(220, 220, 220, 0.8)’
}
    调节空白显示的颜色 这种方式用的rbg三色调
    -backgroundColor 以前js或css就有了的 这不多讲
