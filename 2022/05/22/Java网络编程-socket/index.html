URL类（Uniform Resource Locator） 一致资源定位器的简称，它表示Internet上某一资源的地址
URL的组成 Protocol:resourceNam 协议名指的是获取资源所使用的传输协议，如http\ftp\file\等
例如：public URL(String protocol,String host,String file)
TCP（Transport Control Protoco!）
面向连接提供可靠的流式数据传输协议
URL、URLConnection、Socket、ServerSolet等都可以进行协议网络通信
